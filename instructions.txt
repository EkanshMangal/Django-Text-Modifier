To install django, we use:
    pip install django
Python version: 3.7.4
Django version: 3.0.6
To check Python version:
    python --version
To check Django version:
    python -m django --version

To start new project:
    django-admin startproject mysite    // here mysite is name of project

To run the server or up the website:
    python manage.py runserver  // Run this in the folder where manage.py is located
